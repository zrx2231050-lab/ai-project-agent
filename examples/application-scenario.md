# Xiaomi Token Application Scenario

目标：申请小米模型 token 免费额度，用于增强 AI 项目执行 Agent。

当前进度：
- 已完成本地可运行原型。
- 已实现多 Agent 工作流、RAG 问答、中文工作台和模型 Provider 边界。
- GitHub 仓库已准备好用于申请表审核。

风险：
- 免费额度尚未确认，不能直接依赖真实模型。
- 如果上传资料较碎片化，抽取结果需要来源引用和审核 Agent 复核。

待办：
- 使用小米模型替换本地规则抽取，提高结构化摘要质量。
- 用小米模型增强规划 Agent，生成更细粒度任务拆解。
- 用小米模型增强审核 Agent，识别无来源断言和潜在幻觉。

决策：默认保持本地可运行，拿到 token 后通过环境变量切换到 XiaomiProvider。
